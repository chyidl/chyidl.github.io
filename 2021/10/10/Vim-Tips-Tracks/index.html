Why I Still Use Vim

  • Vim is exists on all Unix systems, support all file formats and programming languages
  • Mouse free navigation
  • Vim is very light-weight and fast.
  • Vim is highly configurable. (~/.vimrc)
  • Memory Usage Low
  • Startup Time short
