持久化

Redis 持久化主要两大机制: AOF (Append Only File) 日志和RDB 快照

  • AOF - Append Only File

    AOF日志是Redis执行完命令，把数据写入内存，然后才记录日志,记录的是Redis收到的每一条命令

  • RDB - 内存快照

    内存快照：指内存中的数据在某一个时刻的状态记录
